Centos7安装zookeeper

2019-12-13 Comments

Zookeeper的安装
（继hadoop安装之后，hadoop的3台机器的HOSTS如下：）
192.168.200.101 master
192.168.200.102 slaver1
192.168.200.103 slaver2

1.master机上操作
（1）上传zookeeper包
本次安装上传到 /opt下
（2）解压
#cd /opt
#tar -zxvf zookeeper….

（3）修改配置文件
#cd /opt/zookeeper-3.4.14
#mkdir data
#cd data
#echo 1 >myid
#cd ..
#cd conf
#mv zoo_sample.cfg zoo.cfg
#vi zoo.cfg
修改:
….
dataDir=/opt/zookeeper-3.4.14/data
….
最后增加：
server.1=192.168.200.101:2888:3888
server.2=192.168.200.102:2888:3888
server.3=192.168.200.103:2888:3888

（3）复制到另二台机器

scp -r /opt/zookeeper-3.4.14/ 192.168.200.102:/opt/

scp -r /opt/zookeeper-3.4.14/ 192.168.200.103:/opt/

  1. 修改myid
    (1) ##slaver1
    #cd /opt/zookeeper-3.4.14/data
    #vi myid
    把1改为2

(2) ##slaver2
#cd /opt/zookeeper-3.4.14/data
#vi myid
把1改为3

  1. 启动zookeeper
    (1)##master

    cd /opt/zookeeper-3.4.14/bin

    #./zkServer.sh start
    (2)##slaver1

    cd /opt/zookeeper-3.4.14/bin

    #./zkServer.sh start
    (3)##slaver2

    cd /opt/zookeeper-3.4.14/bin

    #./zkServer.sh start

  2. 检查
    （1）##master

    cd /opt/zookeeper-3.4.14/bin

    #./zkServer.sh status

（2）##slaver1

cd /opt/zookeeper-3.4.14/bin

#./zkServer.sh status

（3）##slaver2

cd /opt/zookeeper-3.4.14/bin

#./zkServer.sh status

其中有2台机器为： follower,1台机器为leader,这表示正常
